# Community Portal

A web-based Community Portal developed using PHP and MySQL that allows users to interact, manage content, and upload files through an admin-managed system.

## Features
- Admin account creation
- User interaction portal
- File upload system
- Organized project structure
- Backend processing using PHP

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript

## Project Structure
community_portal/
├── assets → CSS, JavaScript, images
├── includes → Database configuration and reusable PHP components
├── public → Public facing pages
├── uploads → User uploaded files
└── create_admin.php → Script to create admin account

## How to Run the Project

1. Install XAMPP or WAMP.
2. Place the project folder in the `htdocs` directory.
3. Start Apache and MySQL from XAMPP.
4. Create a database using phpMyAdmin.
5. Configure database connection in the project files.
6. Open the project in browser:
   http://localhost/community_portal
